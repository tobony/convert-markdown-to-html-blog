# 요구사항 변경 기록 (2025-06-08)

## siteA.html 마크다운 목록/텍스트 레이아웃 개선

### 변경 배경
- 마크다운 복사 시 <ul>, <ol>, <li>, <li><p> 구조에서 줄바꿈/여백 등 레이아웃 오류 발생

- CSS 리셋(normalize 일부) 적용

- 마크다운 복사/붙여넣기 시 목록 및 텍스트 레이아웃 일관성 확보
- 반응형 환경에서도 동일한 결과 보장
