# 마크다운 샘플 파일

이 파일은 애플리케이션이 처음 실행될 때 로드되는 샘플 마크다운 파일입니다.

## 기능 사용법

1. 왼쪽 패널에 마크다운을 입력하세요
2. 오른쪽 패널에서 HTML 결과를 확인하세요
3. 필요한 경우 사용자 정의 CSS를 추가할 수 있습니다

## 마크다운 예시

**굵은 텍스트**와 *기울임꼴 텍스트*를 사용할 수 있습니다.

### 코드 블록

```javascript
function hello() {
  console.log("안녕하세요!");
}
```
